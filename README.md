# Basic Python Utility Projects

This repository contains a collection of 8 simple and unique Python scripts designed for beginners. These projects cover various concepts like loops, functions, system modules, and user input handling.

## 🚀 Projects Included:
1. **Email Slicer**: Extract username and domain from an email.
2. **Expense Tracker**: A simple tool to calculate total daily expenses.
3. **Guess the Number**: A fun interactive number guessing game.
4. **Password Generator**: Generates strong and secure random passwords.
5. **Rock Paper Scissors**: Classic game against the computer.
6. **Simple Alarm**: A countdown timer with a notification message.
7. **System Info**: Displays basic operating system and processor details.
8. **Unit Converter**: Converts distance from Kilometers to Miles.

## 🛠️ How to Run
1. Make sure you have **Python** installed on your system.
2. Clone this repository or download the files.
3. Open the terminal/command prompt in the project folder.
4. Run any script using:
   ```bash
   python filename.py
